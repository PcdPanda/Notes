# 1.术语和定义

### 1.1 座位

##### Small/Big Blind

- 第1/2个入场的玩家,必须下小盲注/大盲注
- 翻牌阶段第1/2个作出决策,风险很高
- 在Stack很少的时候,可以提前作出All In,起到威慑作用

##### UTG (Under the gun)

- 盲注之后靠前的玩家
- 入池风险最高的位置,因为是preflop之后最早决策的

##### MP (Hijack)

- 盲注之后居中的玩家

##### Cutoff

- 倒数第二个玩家

##### Button

- 最后一个玩家
- 信息量最丰富,获胜概率最大

### 1.2 决策

##### Check (过牌)

##### Bet (下注)

##### Call (跟注)

##### Raise (加注)

##### Fold (弃牌)

### 1.3 筹码

##### Stack

- 玩家当前拥有的所有筹码

##### Effective Stack

- 未弃牌玩家中的最大Stack
- <u>自己这轮可能受到的最大损失</u>

##### <u>M-Ratio</u>

- Effective Stack / (盲注+前注)

- 如果每轮都Fold,可以玩的轮数,<u>描述了Blind对玩家的重要性</u>
- Effective-M: M-Ratio * Number of Players

# 2. 策略和分析

### 2.1 基本策略

##### <u>Win Rate</u>

- 后面一直没有获得想要的牌的概率,<u>不能只考虑一轮</u>

##### <u>Pod Odds</u>

- Pod Odds = Call Value / (Call + Bet Value) = 需要下的注 / (需要下的注和所有以下的注)
- 当Pod Odds大于胜率时,收益期望为正

### 2.1 分析

1. 寻找自己不知所措的决策点
2. 分析所有场上玩家的模型
3. 根据场上玩家的表现更改模型
4. 作出决策
