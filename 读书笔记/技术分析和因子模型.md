# 2. 因子模型

### 2.1 因子挖掘

##### 因子挖掘步骤

1. 用数学描述明确问题种类,是预测/分类/最优化
2. 构造计算逻辑,设定一些列可行集合参数
3. 指定明确的评估指标用于选取最合适的因子

##### 信息熵评估法

$$
H(X)=-\sum p(x)\ln p(x)
$$

1. 使用分组/拟合分布来计算因子值的信息熵
2. <u>优秀因子的信息熵应该至少大于0.5</u>

##### 因子优化

| 处理目标           | 处理方法                                                |
| ------------------ | ------------------------------------------------------- |
| 均值归一化         | 使用Max-Min/相对值/Z-score/差分/log相减等方法计算相对值 |
| 尾部处理           | 使用截断/平方/立方根或者tanh/logistic函数来重新映射因子 |
| 确保因子保序       | 使用单调函数处理                                        |
| 即时性和平稳性处理 | 调整计算窗口大小(通常从20出发)                          |

### 2.2 线性模型



##### 模型定义

$$
\arg\min_{\beta}\frac{1}{T}\sum_{t=1}^T(r_t-X^T\Beta)^2+2\lambda\sum_{n=1}^N[\frac{1-\alpha}{2}\beta_n^2+\alpha|\beta_n|]
$$

| $N/n$    | $\Beta/\beta_{n}$   | $T/t$        | $w_t$           | $X/x_{tn}$               | $\lambda$  | $\alpha$          | $r_t$             |
| -------- | ------------------- | ------------ | --------------- | ------------------------ | :--------- | ----------------- | ----------------- |
| 因子数量 | 第$n$个因子的系数值 | 时间序列长度 | $t$时刻样本权重 | 第$n$个因子在$t$时刻的值 | 正则化权重 | 岭回归和lasso比重 | 在$t$时刻的收益率 |

##### 迭代法计算模型参数

1. 定义每个因子的计算中间变量
   $$
   \gamma_n=\frac{1}{T}\sum_{t=1}^Tw_tx_{tn}(r_i-\hat r_i+\beta_nx_{tn})
   $$
   

2. 定义平滑函数
   $$
   S(z,g)=\begin{cases}z-g\quad z>0,g<z\\z+g\quad z<0, g<-z\\0\quad\text{Otherwise}\end{cases}
   $$

3. 获得迭代解
   $$
   \hat\beta_n=\frac{S(\gamma_n, \lambda\alpha)}{\sum_{t=1}^Tw_tx_{tn}^2+\lambda(1-\alpha)}
   $$

##### 大样本计算优化

- 重新定义$\gamma_n$的近似值
  $$
  \gamma_n=\sum_{t=1}^Tw_tx_{tn}r_t-\sum_{i=1}^N\beta_i\sum_{t=1}^Nw_tx_{tn}x_{ti}+\beta_n\sum_{t=1}^Tw_tx_{tn}^2
  $$

- 迭代中当一个参数$\beta_n$在0附近稳定后,直接删除该因子

##### 线性模型生成步骤

1. 使用分层测试/收益率相关性等方法评估因子质量
2. 从所有因子中选择质量最好的,再选择补充性最好的因子
3. 重复2直到获得$N$个因子
4. 添加约束条件并调整$\alpha$,确保模型在每个因子上的暴露均衡,且趋近于0. $\alpha$越小,暴露约均衡,$\alpha$越大,模型越简洁
5. 逐步降低$\lambda$训练模型,并将模型导入到分层测试,获得最优策略

### 2.3 非线性模型

##### 非线性模型转化

1. 将因子值互相相乘,作为新的因子添加到线性模型中

2. 相乘的幂数不超过3,否则因子数量太大
3. 新因子和计算用的成分因子做差,减少相关性
4. 归一化新因子,通常使用Max-Min方法

##### 差分遗传算法

1. 选取一个高质量因子作为$p1$
2. 随机选取两个因子,并对他们做差,获得$d$
3. 随机获得第三个因子,并在这个因子上施加扰动项$d$获得$p2$
4. $p1$和$p2$随机杂交,选取最优的子类作为新的因子

##### 模型评估

| 可解释性 | 计算效率 | 过拟合概率和优化便利度 | 模型参数和表现曲线的平滑度 |
| -------- | -------- | ---------------------- | -------------------------- |

# 3. 交易策略构建

### 3.1 交易策略落地

##### 策略结束强制平仓方案

| <u>限制最长持仓时间</u> | 回测结束清仓 | 临近末尾禁止开仓 | 将持仓动视为连续开平仓操作 |
| ----------------------- | ------------ | ---------------- | -------------------------- |

##### 止损方案

| 收益率止损 | 持仓时间止损 | <u>根据波动率移动止损线</u> |
| ---------- | ------------ | --------------------------- |

##### 策略评估数据采样

| 采样方式             | 缺陷                     |
| -------------------- | ------------------------ |
| <u>每笔交易采样</u>  | 交易数量缺失导致样本不足 |
| 每个数据点都采样     | 数据量太大,计算耗时      |
| 所有数据点降采样     | 会忽略bar内的信息        |
| 只考虑有仓位的数据点 | 无法评估机会成本         |

### <u>3.2 策略检验</u>

##### <u>Bootstrap采样步骤</u>

1. <u>计算策略在$N$个bar上的收益率$\tilde r_t$,并进行去均值化$r_t=\tilde r_t-\bar r$</u>
2. <u>具有重复性地从$r_t$中随机选择$N$个收益率$r_n$并计算均值$\bar r$</u>
3. <u>重复2获得$K$个样本采样均值$\bar r_k$,并构造样本收益率分布</u>
4. <u>计算真实策略收益率$\bar r$在样本收益率分布中的P-value,评估策略质量</u>

##### <u>MCP采样步骤</u>

1. <u>计算策略在$N$个bar上的收益率$\tilde r_t$,并进行去均值化$r_t=\tilde r_t-\bar r$</u>

2. <u>对收益率$r_t$进行标准化,大于0的设置为1,小于0的设置为-1</u>
3. <u>打乱标准化后的收益率,之后和真实bar收益率相乘,并计算乘积后的均值$\bar r$</u>
4. <u>重复3获得$K$个样本采样均值$\bar r_k$,并构造样本收益率分布</u>
5. <u>计算真实策略收益率$\bar r$在样本收益率分布中的P-value,评估策略质量</u>

##### <u>多策略采样</u>

1. <u>使用Bootstrap/MCP对$N$个策略采样获得$N$个收益率均值$\tilde r_n$</u>
2. <u>从$r_n$中选择表现最好的收益率均值作为第一个样本$\tilde r_1$</u>
3. <u>重复1-3获得$K$个样本$\tilde r_k$</u>
4. <u>计算$N$个真实策略收益率在$K$个样本中的P-value,评估策略质量</u>

##### <u>分层测试步骤</u>

1. <u>将数据按照时间序列划分成若干个fold和一部分验证集 (验证集和fold数据中要隔开一段距离,避免使用未来信息)</u>
2. <u>将每个fold分成训练集和测试集(上一个fold的测试集可以是下一个fold的训练集)</u>
3. <u>在每个fold中训练模型参数,并找到最优解</u>
4. <u>分析fold内和fold间各个指标的表现和平稳度,来评估模型质量</u>
5. <u>选择最好的模型在验证集上做测试,测试结果可以作为未来表现的参考</u>
6. <u>根据验证集表现和验证集大小的关系,可以得到模型的更新频率</u>

### 3.3 回测系统问题分析

##### 回测和实盘不一致的原因

| 原因                 | 注释                              | 解决方法                                  |
| -------------------- | --------------------------------- | ----------------------------------------- |
| 回测数据趋势         | 回测使用的数据的平均净收益率不为0 | 回测结果减去这段时间市场收益率            |
| 回测结果是运气导致的 | 策略在当前市场中刚好表现优异      | 使用MC/Bootstrap计算获得当前表现的P-value |
| 选择偏差             | 人们只会选取表现最好的策略        | 扩大数据覆盖范围                          |

- 很多市场数据可能也是随机因素造成的结果,因此信噪比很低

- <u>优秀的策略可以区分数据是来自真实市场还是随机生成的</u>
- 数据抽样的时候,必须保证没有bias,会覆盖所有可能性

##### CSCV法评估回测过拟合 (策略同质化时效果好)

1. 对于$F$组时序数据和$N$个策略,获得$F\times N$的PNL矩阵
2. 从$F\times N$的矩阵中随机抽取$F/2$行,构造大小相同的训练矩阵和验证矩阵$T,V$
3. 在$T$和$V$上计算所有模型性能,获得$N$维向量$R_T$和$R_V$
4. 通过$R_T$找到在$T$中表现最好的模型,并获得该模型在$V$中的表现分位数$r_T$,并计算$\lambda_V=\log\frac{r_V}{1-r_V}$
5. 遍历所有抽取方式,重复步骤$2-4$获得若干个$\lambda_V$样本
6. 通过pdf计算过拟合指标$PBO=\int_{-\infty}^0p_\lambda(\lambda)\mathrm d\lambda$,越大过拟合概率越高
