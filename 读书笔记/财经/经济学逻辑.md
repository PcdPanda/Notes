# 4. 宏观经济学

- 经济活动只取决于消费和预期消费
- 生产的成本包括要素成本和使用者成本,要素成本代表自然和劳动力资源,使用者成本代表资本资源

- 生产成本中的稀缺资源决定了产品的可供给产量
- 微观生产效率描述了行业对资本家的利润率,决定了投资回报率
- 宏观生产效率描述了生产力,由人口素质和制度决定,因为效率乘数,导致宏观生产效率相差巨大
- 投资过热会挤占消费空间,导致总利润减少
- 需要经济增速时,需要将所得分配倾向于投资者,并可以带来就业
- 需要扩大经济发展空间时,需要倾向于劳动者,减小贫富差距,并扩大消费,但经济发展会变慢,且容易带来失业
- 投资风格分为以现金流为目标和低买高卖增值为目标
