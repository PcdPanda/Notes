# 1. 美联储和大萧条

### 1.1 央行和美联储的职责

##### 货币政策

- 根据经济状况反向调整利率
- 利率和隔夜拆解利率会传导到各行各业
- 货币政策具有<u>宏观性</u>,会影响所有行业,使用起来杀伤力太强

##### 流动性供给

- 当银行难以应对挤兑时,提供短期贷款提供流动性
- 短期流动性利率较高,来避免恶意贷款

##### 金融监管

- 减少金融系统性风险
- 和其他机构合作为金融系统提供支持

### 1.2 美联储的金本位制

##### 优点

- 货币信用强,币值和通货膨胀长期稳定
- 国际贸易信用高,有利于吸引投资

##### 缺点

- 短期币值容易受到金矿开采影响,人民容易受到影响
- 汇率和货币政策不灵活,且管理黄金需要成本

### 1.3 1929年大萧条

##### 美联储责任

- 奉行清算主义理论,认为经济和资产中有泡沫
- 为了奉行金本位没有对应调节利率水平
- 对银行挤兑反映不足,认为银行信贷过度,体系收缩才可以健康发展,本质上还是清算主义理论

##### 罗斯福应对方案

- 建立联邦存款保险公司,消除了人民去银行挤兑的动机
- 废除金本位制,允许放松货币政策,结束通货紧缩

##### 经济复苏

- 经济衰退后通常可以伴随一轮更快的复苏

- 过早收紧货币政策可能会带来二次衰退

### 1.4 2008次贷危机

##### 外部因素

- <u>新兴国家需要积累美元外汇储备,对房地产资产的需求激增</u>
- 经济过热,消费者过于乐观
- 银行和证券公司不断降低贷款标准,给不良贷款埋雷 (奇异住房抵押只有在房价上升的过程中借款人才还的起贷款)
- 个人和机构杠杆率过高,管理风险的能力不足以处理复杂的金融衍生品

##### 美联储因素

- 没有准确估计房价下跌对整个金融系统和经济的影响
- 没有严格监管金融公司和产品(房地美和房利美的资本充足率问题)
- 对消费者保护不够,没有提示产品风险
- 缺少对整个金融系统的全方面监管,需要监管从上游到下游的整个链条 (房产商,证券公司,保险公司,证券购买者,贷款人)

##### 总结

- 缓解金融危机的最好方法就是使用宽松的货币政策,并及时提供大量流动性
- 如果没有对整个链条的控制,会导致风险和恐慌以指数级蔓延
- <u>如果系统中存在大而不倒的企业,说明存在根本性缺陷</u>
- 就业=经济>通胀>股市

### 1.5 金融危机的潜在因素

##### 通货膨胀

- 货币政策制定者过于乐观,认为稍高的通货膨胀可以大幅提高就业率

- 稍高的通货膨胀只有短期有效,长期会对经济造成严重的负面影响
- 通货膨胀的根本原因是需求太大,货币太宽松,<u>禁止涨价的政策治标不治本,且会隐藏问题</u>

##### 提高利率

- 提高利率可以有效地收缩经济,降低通胀压力,但是副作用非常强,会导致经济萧条
- <u>应对过热的资产使用针对性的金融工具,例如保证金比例而不是利率,避免牵一发动全身</u>

##### 流动性因素

- 负债的流动性较强,资产的流动性较差,当出现恐慌时会发生挤兑
- 不良资产在整个社会到处流通,给系统增加了极大的不确定性

### 1.6 量化宽松

##### 使用逻辑

- 在利率无法下降的时候依然可以降低市场利率,促进市场发展
- 扩表只增加了银行/政府在美联储账户的储备,但是并没有直接进入流通货币,因此没有进入M2,也不会导致通货膨胀

##### 操作步骤

1. 印钱购买国债,扩大资产负债表
2. 政府获得资金支出
3. 美联储减少市场的国债供应,国债收益率下降,推动市场融资和经济发展

# 2. 价值投资理念

### 2.1 企业价值的意义

##### 基于价值交易

1. <u>准确估计价值</u>
2. 寻找被市场低估和高估的标的
3. 在合理的时间对错误定价的标的进行交易,在安全边际足够的时候要果断进入

##### 衡量价值的因素

| 财力资源 | 人力资源        | 市场资源                  |
| -------- | --------------- | ------------------------- |
| 优质资产 | 技术护城河,专利 | 市场大小和<u>增长潜力</u> |
| 盈利能力 | 管理能力        | 品牌价值                  |
| 现金流   | 人才能力        | 商业模式                  |

### 2.2 市场特征

##### 市场有效性

- 通常市场都是有效的,因此寻找必胜的策略没有意义
- 市场极端乐观和恐慌的时间很少,10年左右出现一次,但这是入场的最好机会,因为这个时候市场无效

##### 市场周期性

- 市场的周期性本质来源人性的波动,而不是基本面的改变
- 泡沫破灭的时候标的<u>会跌90%更多</u>,有参考意义,但是也是买入机会
- 当市场好/差到看起来不像是真的时候,往往就不是真的,要相信自己的判断

### 2.3 投资组合构建

##### 组合构建方式

1. 明确自己可以接受的风险
2. 提高选股标准,将投资范围缩小到自己可以接受的风险范围内
3. 选择好预测/财务透明的公司,并躲开经济衰退期
4. 潜在的选项可能会显示客观不足或者不受欢迎,需要深入分析这种观点是否足够客观
5. 从现存可能性中选择相对最好的(需要考虑未来发展)

##### 风险来源

| 投资工具性质(债券/股票/期货/期权) | 锚定物性质/波动率 | 杠杆使用情况 | 锚定物基本面稳定性 | 系统性风险 |
| --------------------------------- | ----------------- | ------------ | ------------------ | ---------- |

##### 风险正确解读

- 收益和风险不是线性关系,边际效应是递减的
- 风险不能用波动率来衡量,而是实际回撤
- 收益率的分布不是正态分布,而是肥尾的,因此要特别估计
- <u>风险始终存在,没有发生可能只是运气好</u>

# 3. 股票市场分析

### 3.1 宏观货币政策

##### 利率

| 因素       | 解释                                                       |
| ---------- | ---------------------------------------------------------- |
| 市场流动性 | 利率上升导致市场资金总量减少,所有资产价格下降              |
| 贴现率估值 | 利率上升,会降低未来现金流的估值,导致成长股大幅下跌         |
| 风险对价   | 利率上升企业会经营地更加谨慎,减缓增长速度,因此股价下跌     |
| 风险偏好   | 利率上升,本质上提高了股票的仓位比例,因此投资者需要向下调仓 |

##### 企业对宏观政策敏感度

- 收益风险比越高的企业(成长股)对利率和贴现率更敏感
- 稳定的低收益率资产对流动性更敏感

### 3.2 市场情绪

##### 预期和事实

- 预期往往体现在价格中,当事实超预期发展时会造成股价剧烈波动
- 价格震荡说明市场对预期的不一致性越强,在事实出现前适合反转
- 上涨过程中,股价容易向上反映过度,随后回调上涨
- 下跌过程中,股价容易滞后反映不足

##### 机构行为

- 当价格连续超预期利空时,机构会因为回撤管理产生踩踏,此时会<u>导致错误定价</u>
- 基金为了业绩不落后,会抱团参与被大幅超高的热点板块,导致错误定价. 此时可以参与,但<u>必须随时止损</u>
- 机构参与者越多的市场,定价越理性,散户越难获得收益,<u>此时不如根据宏观指标买指数</u>

##### 流动性影响

- 市值占M2比重越小的市场.对流动性越敏感,越容易急涨急跌

### 3.3 标的估值

##### 指数估值

- 指数本质上反映了市场的风险偏好. 当市场风险偏好下降,需要减少仓位时,指数就会下降

##### 个股估值

- 标的的逻辑决定了估值

- 未来前景决定了收益风险比
- 确定性决定了仓位

# 4. 公司逻辑分析

### 4.1 产业逻辑

##### 行业发展

- 行业快速发展时,技术好,擅长把握新产品机会的腰部公司业绩增长最快
- 行业增长停滞,市场收缩,只有龙头能扩张,导致集中度上升

##### 行业经营逻辑

- 重资产: 获得投资,购买资产,再通过规模效应赚钱,业绩稳定
- 轻资产:依靠技术壁垒和运营效率
- 人力型: 看项目资源和管理效率
- 品牌型: 看毛利和渠道控制能力

##### 供应链逻辑

- 当一个公司获得大量投资时,上游供应链利润往往会增长 (对工具,原材料等需求增加)

##### 产品特征

- 消费品公司容易维持强者恒强局面,但是容易受到经济影响
- 高科技公司容易受到技术变革影响

### 4.2 公司逻辑

##### 商业模式

- 连锁型企业需要规模扩张增加盈利,机械复制扩张型的企业增长快

##### 管理模式

- 管理优秀的公司可以灵活转型,可以快速面对竞争和挑战
